Step 1: Open Script Editor

    • Click Spotlight Search in the top-right corner of your screen
    • Type Script Editor
    • Press Return to open it

Script Editor is a built-in macOS app used to run AppleScript code.

Step 2: Create a New Script

    • In Script Editor:
        ◦ Click File → New
    • A blank script window will appear

Step 3: Copy the uptime script

    • Select all of the uptime script code below:
    • Copy it (Command + C)

Step 4: Paste the script into Script Editor

    • Click inside the blank Script Editor window
    • Paste the script (Command + v)

Step 5: Compile the Script (Required)

This step ensures the script has no errors before running it.

    • Click the Compile button (hammer icon at the top)
    • Watch for:
        ◦ No errors → Continue
        ◦ Errors → Do not run the script.

Step 6: Run the Uptime Script

    • Click the Run button
    • Review the confirmation window that appears:
        ◦ It will list all Uptime find
